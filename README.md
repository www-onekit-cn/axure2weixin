# 微信小程序版字节跳动小程序SDK

## 项目介绍  
此项目是为了广大开发者能够快速地在微信小程序使用字节跳动小程序的组件和API。
## 特色功能
**1.** 官方标准  
**2.** 全套功能  
**3.** 全套组件  
**4.** 无学习门槛  
**5.** 支持第三方NPM库  
（如有其它问题，请加技术支持QQ群：185654475）
